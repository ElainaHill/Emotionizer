# Benedict

Benedict is an interactive Jupyter Notebook-based project for experimenting and iterating on data, analyses, and small prototypes. The primary notebook in this repository is [Benedict_2_0.ipynb](Benedict_2_0.ipynb), which contains the main code, visualizations, and narrative for the project.

**Quick overview**
- **Type:** Jupyter Notebook project
- **Primary file:** [Benedict_2_0.ipynb](Benedict_2_0.ipynb)

**Getting started**
1. Install Python 3.9+ (or your preferred compatible version).
2. Create and activate a virtual environment:

	```bash
	python -m venv .venv
	source .venv/bin/activate
	```

3. Install dependencies (if a requirements file is provided) or install Jupyter:

	```bash
	pip install -r requirements.txt  # if present
	pip install jupyterlab notebook
	```

4. Open the notebook:

	```bash
	jupyter lab    # or `jupyter notebook`
	```

5. Open [Benedict_2_0.ipynb](Benedict_2_0.ipynb) in the Jupyter interface and run the cells interactively.
